Testing the LAMP Stack.

How to test your current LAMP stack to make sure that everything is working properly.

  1. Verify Apache:
    Open a web browser and enter http://localhost in the address bar. If Apache is installed, you should see the default welcome page.

  2. Test PHP:
    Create a new PHP file in the document root directory of your Apache virtual host. For example, create a file named info.php with the following content:
    <?php
phpinfo();
?>
    Save the file and access it in your web browser (e.g., http://your_domain.com/info.php). If PHP is working, you should see a page displaying PHP info.

  3. Check MySQL:
    Open terminal and log in to the MySQL shell: sudo mysql. Once logged in, run this following command to all databases:
    SHOW DATABASES;
    If MySQL is working, you should see a list of databases.

  4. Create a Test Database:
    While still in the shell, you can create a test database to further verify MySQL functionality: CREATE DATABASE test_db;. Verify that the database is created by running: SHOW DATABASES;
    You should see the database in the list.

  5. Test Connectivity:
    You can now test database connectivity by creating a PHP script that connects to MySQL and selects data from a table in the test database. If the script executes, it indicates that PHP can communicate with MySQL successfully. Congratulations!
