Apache Installation.

How to install Apache2 on Linux.

  1. Open Terminal.
    On Linux, you do this by pressing Ctrl + Alt + T or searching for it in the Applications menu.

  2. Update the package index.
    This is not really needed, but it will make everything updated and on the latest version, which would avoid possible issues. To update the index, run this in the terminal:
    sudo apt update and enter your password.

  3. Install Apache.
    Once everything is updated and ready to go, run: sudo apt install apache2 and press "Y" when prompted. This will install all of Apache's files onto your drive.

  4. Verify that Apache is installed.
    Once Apache has been installed, run this in a terminal to make sure it was installed correctly: sudo systemctl status apache2. If everything is good, you should see an output that indicates everything is ready to go!

  5. Access the Apache demo page.
    Finally, open a browser and type http://localhost or just simply "localhost". Congrats! Apache is now ready to be used!
