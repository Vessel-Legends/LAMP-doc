PHP Installation.

How to install PHP v8.2 on Linux.

  1. Add the PHP repo.
    Unlike on Windows, you have to use the Linux package index to install PHP 8.2. Open the terminal and run these 2 commands:
    sudo add-apt-repository ppa:ondrej/php
    sudo apt update

  2. Install PHP.
    Once the repo is added and ready, you can actually install PHP and it's extensions. To do that, run sudo apt install php8.2 php8.2-common php8.2-cli in the terminal and press "Y" when and if prompted.

  3. Verify the installation.
    After everything is installed and ready, it's time to verify if everything worked. Run php -v. You should see the output that shows the version of PHP.

  4. Install PHP extensions (optional).This step is 100% optional so you dont have to do this step, but if you want to install extensions, using the terminal, type: apt search php8.2-, and once you find the extension you want, type sudo apt install php8.2-{extension_name}. Congrats! PHP has been installed!
